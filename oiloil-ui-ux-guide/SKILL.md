# OilOil UI/UX Guide

A modern, clean UI/UX guidance and review skill.

## Installation

```bash
npx skills add oil-oil/oiloil-ui-ux-guide
```

## Modes

- `guide`: Provide compact principles and concrete do/don't rules for modern clean UI/UX.
- `review`: Review an existing UI and output prioritized P0/P1/P2 actionable fixes.

## Features

- Task-first UX with clear primary action hierarchy
- CRAP visual hierarchy (Contrast/Repetition/Alignment/Proximity)
- Cognitive load control and progressive disclosure
- HCI laws (Fitts, Hick, Miller) and interaction psychology
- State coverage: loading, empty, error, success, permission
- Modern minimal style enforcement
